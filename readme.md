# Tic Tac Toe Game ❌⭕🎮

A simple console-based Tic Tac Toe game developed using Java.

## About the Project

This project is a basic implementation of the classic Tic Tac Toe game where two players take turns marking X and O on a 3x3 grid. The game checks for a winner or a draw after each move.

## Features

* Two-player mode (Player X and Player O)
* Console-based gameplay
* Input validation for moves
* Win condition checking (row, column, diagonal)
* Detects draw condition
* Simple and beginner-friendly Java logic

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/rawoof28/Tic-Tac-Toe-Game.git
   ```

2. Open the project in any Java IDE (Eclipse, IntelliJ, VS Code).

3. Compile the file:

   ```bash
   javac TicTacToe.java
   ```

4. Run the program:

   ```bash
   java TicTacToe
   ```

## How to Play

1. Run the program.
2. Player X starts first.
3. Enter row and column numbers to place your mark.
4. Players alternate turns.
5. The game ends when a player wins or the board is full (draw).
